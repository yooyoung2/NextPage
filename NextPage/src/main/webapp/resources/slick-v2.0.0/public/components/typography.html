You have three sets of font-family available to use.

Raleway
Roboto
Monospace

Headings

Slick has all the html headings, <h1> through <h6>, styled for you. Also you can use .h1 through .h6 classes when you want to match the font styling of a heading but still want your text to be displayed inline.
